LPToolKit Settings

This is the configuration section. The navigation bar above provides access to all settings for this software. Details about what each screen does is outlined below.

OSC to MIDI

Converts OSC to MIDI messages for connected MIDI devices.

Device Manager

Controls which MIDI devices are to be treated as pad devices, and knob devices.

Implants

Provides a list of loaded implants, which are currently active, and what area of hardware devices they are mapped to.
